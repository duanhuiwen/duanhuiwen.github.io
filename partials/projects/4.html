4Homework

2012.9 - 2012.12 | Espoo, Finland

Key words: Android, Education, PHP & MySQL

This app helps student remember their homework, the student on duty (for example sharing the homework with classmates for one week) will learn responsibility.

Project goals:

  • Mobile Client:
    • List of the upcoming homework.
    • Clicking on the homework item will slides to detail view.
    • Student will be able to set the class ID and server address.
    • Every newly added homework will be pushed to mobile client.
  • Web Client:
    • Login required on admin page.
    • List all recent homeworks assigned based on the class ID.
    • Add/delete/modify homework.
